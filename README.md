# DAT300 – Applied Deep Learning

This repository contains coursework and experiments from the course **DAT300 – Applied Deep Learning** at the Norwegian University of Life Sciences (NMBU).

The purpose of this repository is to demonstrate practical work with modern deep learning techniques using Python and Jupyter notebooks.

## Contents

- **CA1_ANN.ipynb**  
  Implementation of Artificial Neural Networks (ANN) for classification tasks.

- **CA2_CNN.ipynb**  
  Convolutional Neural Networks for image classification and feature extraction.

- **CA3_Segmentation_U-Net.ipynb**  
  Image segmentation using the U-Net architecture.

- **CA4_Auto_Encoders.ipynb**  
  Autoencoders for representation learning and dimensionality reduction.

- **CA5_RNN_Transformers.ipynb**  
  Sequence modeling using Recurrent Neural Networks and Transformer-based architectures.

## Technologies Used

- Python
- Jupyter Notebook
- TensorFlow / Keras
- NumPy
- Matplotlib
- Scikit-learn

## Learning Objectives

Through these assignments, the following concepts were explored:

- Neural network architecture design
- Image classification with CNNs
- Image segmentation with U-Net
- Representation learning with autoencoders
- Sequence modeling with RNNs and Transformers
- Model training, evaluation, and visualization
